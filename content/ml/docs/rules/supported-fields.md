# ആമുഖം

ഫാൽക്കോ പിന്തുണക്കുന്ന ഫീൽഡുകൾ ഇതാ. ഈ ഫീൽഡുകൾ ഒരു ഫാൽക്കോ നിയമത്തിൻറെ `condition` കീയിലും `output` കീയിലും ഉപയോഗിക്കാൻ കഴിയും.   [`json_output`](../../alerts#json-output) എന്നത് `true` ആയി സെറ്റ് ചെയ്യുമ്പോൾ, നിയമത്തിൻറെ `output` കീയിൽ ഉൾപ്പെട്ട ഏത് ഫീൽഡുകളും അലേർട്ടിൻറെ `output_fields` ഓബ്ജക്റ്റിലും ഉൾപ്പെടുന്നു. 

നിങ്ങൾക്ക് ഈ ഫീൽഡുകളുടെ സെറ്റ്, `falco --list=<source>` എന്നത് വഴി `<source>` എന്നത് താഴെയുള്ള ഉറവിടങ്ങളിലൊന്നായിട്ടും കാണാവുന്നതാണ്.

# സിസ്റ്റം കോളുകൾ (ഉറവിട

#  `syscall`)

`syscall` ഇവൻറ് ഉറവിടഫീൽഡുകൾ  [kernel module](/docs/event-sources/drivers/) ആണ് ലഭ്യമാക്കുന്നത്. ഈ ഫീൽഡുകൾ സിസ്ഡിഗ് കാപ്ച്ചറുകൾ ഫിൽട്ടർ ചെയ്യാൻ ഉപയോഗിക്കാവുന്ന [Sysdig filter fields](https://github.com/draios/sysdig/wiki/Sysdig-User-Guide#all-supported-filters) എന്നതിന് സമാനമാണ്.

```
# System Kernel Fields
$ falco --list=syscall
```
