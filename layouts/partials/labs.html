Lab 1 - Introduction to Falco

General introduction to Falco in single Linux hosts. Learn how to install Falco with Docker, and detect threads like:

  • Container running interactive shell
  • Unauthorized process
  • Write to non user-data directory

This lab is the best place to start your Runtime Security jurney!

Lab 2 - Falco forensics in K8s

In this lab, you can learn how to deploy Falco in a a running Kubernetes cluster and how to define your first custom rules file.

Play the both the intruder & sysadmin roles and detect a Rogue attack

Lab 3 - Falcosidekick: Thread Response Engine

Falco is not just about Runtime Detection, you can perform automated actions to mitigate threads automatically.

In this lab you will learn how to deploy Falco with Falcosidekick and Kubeless. Stop a running pod when you detect with Falco a security thread!
